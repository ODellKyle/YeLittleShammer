Ye Little Shammer
Developed by: Kyle O'Dell

See Bibliography.pdf located in the Bibliography_Reference for credit 
to assets, music, and code inspiration.

CS583 Spring 2020 2D game

User can access how to play from the settings of the game.

How to play:
W – Jump
W(x2) – Double Jump
A – Move left
D – Move right
S(Hold) – Fall through platforms
Arrow Keys(Hold) – Shoot gun


Design:
-2D platformer
-Camera moves when player leaves the edge of the screen
-Enemies have half second delay of attack
-Utilized polymorphism to create unique characters based around
an abstract class.
